#Student Portal Web Application
#Description

A web-based Student Portal that allows students to sign up, log in, and manage their course selections through a secure and user-friendly interface.
The application follows a clientâ€“server architecture with separate APIs for students and courses, ensuring clean structure and scalability.

The system features a modern dark theme UI and dynamic data handling using AJAX for a smooth user experience without page reloads.


---
#Features

Student registration and authentication

Secure login & logout

View available courses

Select and manage enrolled courses

RESTful API for Students

RESTful API for Courses

Dark theme responsive interface

Asynchronous requests using AJAX



---
#Technologies Used

Frontend: HTML, CSS, Bootstrap, JavaScript

Backend: Python, Flask

Database: SQLite

Communication: AJAX, RESTful APIs



---
#Architecture

Students API: handles authentication and student data

Courses API: manages course data and enrollment

Frontend consumes APIs using AJAX for dynamic updates



---

#Purpose

This project demonstrates backend development skills including API design, authentication, database management, and frontend-backend integration.
